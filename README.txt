CONTROLES DE JUEGO:
 · ‘1’ : Inicia una nueva partida.
 · ‘2’ : Carga una partida empezada y guardada anteriormente.
 · ‘3’ : Guarda una partida que tiene que ser iniciada antes de ser guardada.
 · ‘4’ : Sale del juego.

Dentro de la partida:
 · ‘m’ o ‘M’ : Pausa el juego y vuelve al menú.
 · ‘v’ o ‘V’ : Permite volver a introducir un movimiento.
 · ‘0’ : Representa el mazo.
 · ‘12’ : Representa el descarte.
 · ‘1’, ‘2’, ‘3’, ‘4’, ‘5’, ‘6’, ‘7’ : Representan las distintas columnas.
 · ‘8’, ‘9’, ‘10’, ‘11’ : Representan las distintas salidas.
